Notable Assignments and Projects

CirclE

CirclE aims to be a driving force for environmental sustainability by simplifying the process of purchasing eco-friendly products and enhancing the accessibility of recycling options for users.

Community Connect

Community Connect is a platform designed to strengthen local communities by facilitating easier connections between volunteers and community-based projects or events.

Task Master

Task Master is a web-based task management application designed to streamline task tracking and improve productivity for individuals and teams. Built using Ruby on Rails and Bootstrap, it offers a responsive user interface and a robust set of features to manage tasks effectively.

Sum of Squares Calculator

A simple C++ program to calculate the sum of squares of numbers from 1 to a specified number using recursion.
